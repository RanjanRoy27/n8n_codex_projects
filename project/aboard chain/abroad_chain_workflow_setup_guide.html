Workflow ID: 0faiT7g7k8PI3kVI

Current validation: Valid, 0 blocking errors, 22 nodes, no Code nodes.
Purpose: Generate a daily content topic, Bangla caption, Canva visual brief, QA approval, Google Sheets content calendar rows, engagement log template, and a human Meta Business Suite handoff.
The times in node names are SOP stage labels only. They do not pause the workflow until those times. The workflow starts at 9:00 AM by schedule and then runs through the whole content production pipeline.

Required Google Sheet Structure

Create one spreadsheet and add the following tabs. Column names must match the append mappings if you want clean rows.

Content Calendar

Date | Brand | Status | Facebook Time | Instagram Time | Location Tags | Final Pack | Meta Handoff

This tab receives final approved content, revision rows, and tomorrow planning rows.

Performance Log

Date | Platform | Top Post Topic | Reach | Engagement Rate | Lead Clicks | Comments | DMs | What Worked | Notes

This is manually maintained for now. The workflow reads it to simulate the daily Meta Business Suite audit without requiring Meta API credentials.

Lead Log

Date | Task | Lead Name | Phone | Location | Source | Status | Next Follow Up | Notes

This tab receives a daily engagement task template. The human marketer can add real inquiries after replying to comments and DMs.

Credential Setup Checklist In n8n

  1. Create or select a Google Sheet with the three required tabs.
  2. In n8n, create/connect a Google Sheets credential for the client account.
  3. Open every Google Sheets node and select the same credential.
  4. In Brand Config, replace PUT_GOOGLE_SHEET_ID_HERE with the real spreadsheet ID.
  5. In n8n, create/connect an OpenAI credential.
  6. Open each OpenAI node and select the OpenAI credential.
  7. Run Manual Test Trigger.
  8. Check that rows are appended to Content Calendar and Lead Log.
  9. Review the final content pack before enabling the schedule.
  10. Activate the workflow only after the manual run succeeds.

Known Limitations Of This Version

  • No auto-posting to Meta. A human schedules in Meta Business Suite.
  • No direct DM/comment reading from Meta. Lead logging is a Google Sheets task template.
  • No Airtable or Notion. The client requested Sheets for now.
  • Public Meta Ad Library page scraping may be unreliable. It is included only as a free/no-auth signal.
  • Google Trends unofficial endpoint may change or rate-limit. It is included because the user requested free/no-auth sources first.
  • AI output is validated by prompt and IF logic, but a human should still review content before posting.
